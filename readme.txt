The Simplest way to place an intelligent advertising banner to your site with lots of customization option.

== Description ==
This plugin is to place an advertising banner to your blog with title and description of the ad. Also you can customize the ad for particular any post of your blog and the main attractive feature of this plugin is that it will fix the ad to the top of the site while scroll down to the bottom of the site after its position at the sidebar.

It is a plugin that enables individuals and small businesses to promote their products and services through their amazing blogs.

We will add click report for ads very soon, so please if you like this plugin don't forget to rate the plugin to encourage us to ad more cool features to this plugin.

**Features**

* Easily add an image ad with its title and description.
* Customize the ad per post option
* Ad fixes to the top of the page while scrolling down to the bottom of the page after its position at sidebar.

== Installation ==
1. Unpack the `download-package`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. See the details under Admin Panel `Settings -> Ad Box`.
5. Go to `Appearence -> Widget` then add "Ad Box" widget to the sidebar. 
6. Fill the details and done, you are good to go.
7. Please see FAQ for more.

== Frequently Asked Questions == 

= How can I get Ad Box Plugin to blog's sidebar? =
* Go to `Appearance -> Widgets -> Ad Box` and add it into Sidebar.

= What is the best place for the widget? =
* Sidebar bottom would be the best place for the widget, because while scrolling down to the bottom it will fix itself to the top of site.
